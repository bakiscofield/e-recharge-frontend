# PWA Screenshots

Ce dossier contient les captures d'écran de l'application pour le manifest PWA.

## Tailles requises

### Mobile (narrow form factor)
- **Taille recommandée**: 540x720 pixels
- **Ratio**: 3:4
- **Fichier**: `mobile-1.png`

### Desktop (wide form factor)
- **Taille recommandée**: 1280x720 pixels
- **Ratio**: 16:9
- **Fichier**: `desktop-1.png`

## Comment créer les captures

1. Ouvrir l'application dans le navigateur
2. Naviguer vers la page principale
3. Ouvrir les DevTools (F12)
4. Activer le mode responsive (Ctrl+Shift+M)
5. Définir les dimensions appropriées
6. Prendre une capture d'écran
7. Sauvegarder dans ce dossier

## Outils recommandés

- Chrome DevTools Screenshot
- Firefox Screenshot
- https://www.screely.com/
- https://screenstab.com/

## Notes

- Les captures doivent montrer l'application en action
- Éviter les données sensibles ou de test
- Utiliser des données réalistes mais anonymisées
- Format PNG recommandé pour la qualité
